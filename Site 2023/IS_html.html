Sortarea prin inserție

Mod de funcționare

Această metodă de sortare se bazează pe inserarea repetată a elementelor în secvența din stânga acestora astfel încât aceasta secvență să fie ordonată.

Să spunem că avem un vector V[] cu n elemente.

Parcurgem vectorul cu indicele i. Dacă secvența 0, 1, ... i - 1 este ordonată atunci putem insera elementul V[i] in această secvență încât să fie ordonată secvența cu indicii 0, 1, ... i - 1, i.

Luăm pe rând fiecare element V[i] și îl înserăm în secvența din stânga sa.

După inserarea ultimului element din vector, întreg vectorul este ordonat.

Complexitate

Complexitatea acestui algoritm este una pătratica (O(n2)), dar avantajul acestuia este ușurința de transpus în cod, fiind util pentru seturi mici de date.
